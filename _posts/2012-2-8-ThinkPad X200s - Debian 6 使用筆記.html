我已將 ThinkPad X200s 上安裝的作業系統從 Ubuntu 10.04 換成 Debian 6 squeeze。

硬體

WLAN - Realtek RTL8192SE driver

我的 ThinkPad X200s 配裝的 WLAN 網卡是 Realtek RTL8192SE， 參考「Debian 6 and Realtek RTL8192SE WiFi device」。

Express card 與 eSata
Silimage eSata Express card

我有一張 Silicon Image eSATA Express Card，以使用外接的 eSata 硬碟。

狀況: Express card 不能熱插拔。Debian 啟動之後，再插入 Express card 時，不會有任何反應。

解決方式: 修改 /etc/modules ，加入 pciehp 與 sata_sil24 兩個模組。pciehp 是讓 ThinkPad 上的 Express 插槽熱插拔用的； sata_sil24 是 Silicon Image eSATA Express Card 用的。

備註: Linux kernel 版本 2.6.32-5-amd64。先前安裝 Ubuntu 10.04 時，這張 Express card 不需任何設定就可以熱插拔。我不知道 Debian 6 的 kernel 套件是差了哪些項目，必須自己手動加入上述兩個模組。

軟體

CJK 字型別名設定

參考「利用 X 字體別名，讓 Linux 桌面環境顯示文件中的標楷體、細明體與黑體」。Ubuntu 桌面有附加一份 30-cjk-aliases.conf 的字型別名設定文件，但我在 Debian 6 沒找到。所以我從 Ubuntu 桌面複製了一份過來，再額外加上我偏好的設定。

我目前使用的 30-cjk-aliases.conf 的點擊此連結下載: [30-cjk-aliases.conf]。(F**k，本來放在我的 filesonic 空間內，現在不能用來分享了)

Empathy

狀況: 不能新增 MSN 聊天帳號。

解決方式: 安裝 telepathy-haze 套件。不是裝 telepathy-butterfly 。

1. 我不想為了升級某些軟體，卻要升級整個系統。
例如某軟體在 Ubuntu x.04 中的版本是 1.0；如果你想用 1.1 版，通常就要整個 Ubuntu 升級到 x.10 或 y.04 。

2. 我覺得 Debian 的軟體版本較穩定，升級較平順。

3. 我不喜歡 Ubuntu 新版的 Unity 桌面。

4. 從 Ubuntu 9, 10, 11 來看，它的系統架構經常在改變，甚至 .04 和 .10 之間就會有差異。
我不想每半年就要重新調整一次這些小細節。

如果你屬於把作業系統裝下去就按照預設狀態或操作方式一路學習的使用者，那麼 Ubuntu 這些更新是好事。這就像大部份 Windows 用戶的使用經驗也是完全按微軟的規劃走。XP 的介面要他這麼操作，他就這麼操作；升級到 Widnows 7 之後，又按 Windows 7 的介面設計原則再學一套新的操作習慣。

Debian 比較像是工程版，我很容易按照自己的操作習慣訂製整個環境。
我最早使用 Linux 時，用的是 Slackware 版。那是一個鬆散且客製性極高的版本。
延續同樣的使用習慣，所以我就偏好 Debian 。
