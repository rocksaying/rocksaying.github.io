--- title: 我與小黑(ThinkPad X200s) category: computer old-category: 電腦技術 tags: [ubuntu,ThinkPad] ---

我在過年前買了生平第一台筆記型電腦。打從一開始，我就不打算使用 Windows 作業系統，所以我特地只選擇不附OS的機種來看。而我的第二個選擇偏好是，不要觸控板。我一點都不喜歡用觸控板控制指標，又不想要多帶一個滑鼠，在這個條件下，我唯一的選擇就是 TrackPoint (小紅點)，它可以讓我的雙手完全不用離開鍵盤，就能完成所有操作。雖然不附OS的機種很少，又要滿足第二個偏好更難，但非常幸運地，在那極少數可選擇的機種中，偏偏就有滿足我偏好的機種，那就是 ThinkPad X200s 。俗稱小黑。

我拿到小黑後，馬上就開始安裝 Ubuntu Desktop 9.10 。安裝過程非常容易，先下載 Ubuntu Desktop 9.10 的 CD image ，然後解開複製到 SD 記憶卡上。就可以在 boot order 選單中選擇 USB HD ，利用 SD 記憶卡啟動 Ubuntu 安裝程序。已經在使用 Ubuntu 的用戶，可以從工具列執行「系統」→「管理」→「USB啟動碟產生器」，利用這隻小工具將 Ubuntu install CD 的內容轉移到 SD 記憶卡。或者，請參考 將LiveCD移植到USB隨身碟 的步驟。

整個 Ubuntu 安裝過程非常順利，啟動後，所有設備全都找到了，完全沒有碰到任何硬體驅動問題。反而我在安裝 Windows XP 時碰到一堆硬體驅動程式的問題。直到現在，我的 Windows XP 環境的硬體裝置列表中，都還有2個未知裝置顯示驚嘆號。這個結果真是令我感到意外。

在安裝過程中發生個小插曲。我第一次安裝時，選擇的是 Ubuntu Desktop 9.10 x86 版，亦即 32bit 版。啟動後一看，我的 4G 記憶體，作業系統只用了 2.9G 記憶體，卻分配了 1G 記憶體給顯示卡。心想這真是太誇張了。結果上網一查，才想到這是 32bit OS 的記憶體定址限制導致的結果，使得作業系統實際上只能用 2.9G 記憶體，而它定址不到的部份，就全部判定為分配給顯示卡。於是我又改裝 Ubuntu Desktop am64 版，這次的記憶體分配結果就對了，作業系統抓到 3.75G ，分給顯示卡 256MB 。

目前為止，我的小黑幾乎能完成所有我原本在桌機上進行的工作。GOD! 它的鍵盤真好按，我都妄想把它的鍵盤拔下來給桌機用了。只有兩件事是桌機才能做的，那就是玩魔獸世界以及看動畫。

魔獸世界就不多談了。為什麼看動畫也只有桌機才能做呢？第一，因為我的桌機用的是 Lenovo LX220 1920x1200 MVA 面板的螢幕，光是解析度與色彩豐富度，就不是筆記型電腦的 TN 面板比得上的。第二，我用的雖然是整合型主機板，但板上的繪圖整合晶片 (Ati HD3200) 剛好支援 Mpeg4 硬體解壓縮。我用 Windows XP + + 最新版 DirectX 9 + MediaPlayer Homeciname 就能硬解 1080P 影片。播放時的 CPU 使用率還不到10%。不用硬解時， CPU 使用率衝到70%左右，而且畫面變成了連續投影片，根本看不下去。

說到這，要順便提一下 Lenovo 不久前剛推出的新機種 X100 。基本上，我覺得那台的規格與外形，都比 X200s 差，但是它的繪圖晶片也是 Ati HD3200 。所以 X100 那台可以用硬解播放 1080P 影片 (雖然螢幕解析度放不下)，這大概是 X100 最明顯的優點吧。

現在，我的 ThinkPad X200s 加 Ubuntu 9.10 上搭配的應用軟體有:

  1. Firefox: 從 Windows 到 Ubuntu ，一致的選擇。
  2. VirtualBox-OSE: 可以開啟全虛擬化功能，啟動3台VM (各配512MB記憶體)。
  3. 使用 wine + foobar2000 聽音樂: foobar2000 支援 cue + ape/flac 播放。就規格支援度來說， foobar2000 還很難找到替代品。而且我不喜歡用 Rhythombox。
  4. aMule: eMule 的同胞兄弟，使用介面幾乎一模一樣。
  5. wine + uTorrent: 嗯... 因為花園的種子都是 magnet 的。 Ubuntu 內建的 BT client (Transmission) 還不支援這種格式。雖然 Transmission 官網上的最新版已經開始支援 magnet 了，不過 Ubuntu 的 repository 還沒更新。(Ubuntu 10.04起，Transmission 支援 magnet，我不用再裝 uTorrent了)
  6. 內建的媒體播放器還不錯，但不能掛字幕。幸好，VLC media player 可以取代。
