--- title: 入手小紅點鍵盤, IBM Space Saver II category: computer old-category: 電腦技術 tags: [小紅點,trackpoint] ---

前幾天在網拍上買了一隻 IBM Space Saver II 鍵盤。 雖然它似乎已經有10年歷史了，但功能仍然很正常，最重要的小紅點(TrackPoint)依然可以使用。 我現在把它放在公司使用，寫程式時雙手完全不必離開鍵盤，真的很方便。 滑鼠已經被我扔到一邊去了。

全照圖與 TrackPoint 特寫。

將它與我的鍵盤並列。可以看出少了 IBM Space Saver II 少了數字鍵，長度只有一般鍵盤的80%。

將它與我的 ThinkPad X200s 並列。 IBM Space Saver II 屬於舊款鍵盤，它的按鍵排列方式像是一般的80%鍵盤，下方也沒有多出一塊托板。 而新款式的 TrackPoint 鍵盤，外觀就像是筆記型的鍵盤。

驅動篇

這隻鍵盤的接頭是兩個 PS/2 插頭，一個插鍵盤座，另一個插滑鼠座。 如果作業系統不支援 TrackPoint 的話，基本上會被作業系統視為一個鍵盤與一個滑鼠。 此時將無法享受到小紅點的特殊功能，例如中鍵滾輪、Press to Select 。 要使用這些特殊功能，就必須有驅動程式。

Ubuntu Linux 與 Windows 7 內建支援

Ubuntu Linux 內建項目支援這隻鍵盤，不需要另外尋找驅動程式。 TrackPoint 的設定方式與筆電上的一模一樣，Ubuntu/Linux 的使用者請參考本人之《我與小黑之小紅點》。

我沒試過其他的 Linux 桌面版本，但想必其他版本也都沒有問題。

至於 Windows 7 的部份，我是聽賣主說 Windows 7 內建支援。 我本人沒有 Windows 7 系統，所以無法證實。

Windows XP 需手動安裝

Windows XP 預設狀態下，只將此鍵盤的 TrackPoint 視為一般的 PS/2 滑鼠。 在 Windows XP 上需要安裝驅動程式才行使用。 然而 Lenovo 官方網站上，也只提供到 Windows 2000 的驅動程式下載。 儘管官方網站提供的驅動程式不能在 Windows XP 下自動安裝，但是在 Windows XP 還是可以透過手動更新的方式，強制安裝上去，也能正常使用。 我的安裝步驟如下:

第一步、接上 Space Saver II

首先，先將鍵盤上的兩個 PS/2 插頭插入電腦主機上正確的插座上。然後再開機進入 Windows XP。 若已經啟動 Windows XP 了，那就需要重新啟動，系統才能辨識到一個 PS/2 滑鼠設備。

如果沒有問題的話，現在就有一個 PS/2 鍵盤和 PS/2 滑鼠可以操作。 接著就需要安裝驅動程式，啟用 Trackpoing 的神奇魔力。

第二步、取得驅動程式

開啟此連結: TrackPoint IV Keyboard - Files， 下載 w32tp.exe。這就是 Space Saver II Keyboard 的驅動程式。 如果此連結無法開啟或是內容不存在，請搜尋 TrackPoint Keyboard 或 Space Saver II。

下載後將之解壓縮。你可以試著執行 setup.bat 或 SETUPDI.EXE，但保證會失敗。 重點在下一步。

第三步、手動更新驅動程式

開啟控制台的裝置管理員，從「滑鼠與其他指標裝置」的分類下，找到 PS/2 滑鼠。 如果你沒有接上其他滑鼠的話，那麼那個「PS/2 滑鼠」就是指鍵盤上的 TrackPoint。 開啟那個項目，選擇「驅動程式」頁，點擊「更新驅動程式」，選擇從磁碟上指定檔案的方式， 讓作業系統安裝我們第二步時解開的那些檔案。

安裝程式會提示你驅動程式的版本太舊，請繼續安裝。 在某些環境下，安裝程式在100%進度後不會自行結束，而是一直停留在進行中的狀況。 根據我的經驗，請等待大約三分鐘後，直接選擇重新開機。不必懷疑。 重新開機後，系統就會正確地啟用 TrackPoint 功能了。

更新後，你的裝置管理員就會正確顯示"IBM PS/2 TrackPoint"。

在右下角的工具列上會出現小紅點的圖示。點擊它，選擇「TrackPoint Properties」就可以叫出設定畫面， 真正享用 TrackPoint 的神奇魔力了。

手頭寬裕的人，還是直接考慮買一隻一新款式的 ThinkPad TrackPoint 鍵盤吧。 個人強烈推薦無觸控板的最新型號。
