--- title: AXIMCom MR-105NL 接在網路分享器後方的用法 category: computer old-category: 電腦技術 tags: [network,routeros] ---

我今天買了台 AXIMCom MR-105NL 的無線網路分享器，主要是看在價格便宜還支援 3G 網路分享。只是我的用法有點特別，不是它原來預設的方式，所以我調整了一些它的設定。

AXIMCom MR-105NL 的預設工作模式是網路分享器暨 AP 。但我原先已經有一台 MikroTik RouterBoard RB750G 網路分享器，它的功能更穩更好。所以我不需要使用 MR-105NL 的網路分享器功能。我只需要 MR-105NL 接在 RB750G 上，做一台單純的乙太網路集線器暨 AP 。各網路設備的連接情形如下所示。

手機、筆電 -> Wi-Fi 訊號 -> MR-105NL -> 網路線 -> (DHCP) RB750G -> ADSL Modem -> Internet

調整設定

為滿足我的需求，我需要調整 MR-105NL 的設定。關閉 MR-105NL 的 DHCP 功能，並指派 MR-105NL 的內部網路 IP 。

第一步、先用一條網路線，直接連接電腦與 MR-105NL。注意，只要電腦與 MR-105NL 兩台連接， MR-105NL 先不要接其他交換器或分享器。還有一點，網路線要接在 MR-105NL 的 LAN 埠上。接在 WAN 埠就進不了 MR-105NL 的管理介面。

按 MR-105NL 的預設模式，你可以直接透過 Wi-Fi 連接 MR-105NL 並進入它的管理介面。然而 Wi-Fi 連線需要配合一個 DHCP伺服器，但我卻要關掉 MR-105NL 的 DHCP伺服器，這樣 Wi-Fi 就暫時不能用。所以還是拿條網路線連接 MR-105NL ，才方便後續的設定工作。

第二步、電腦上開啟瀏覽器，網址輸入 http://192.168.1.1:8080/ ，就可以進入 MR-105NL 的管理介面。192.168.1.1 是 MR-105NL 的初始 IP。預設帳號與密碼皆為 admin 。

基於安全考量，建議先變更 MR-105NL 的管理密碼。

第三步、我要用 RB750G 管理 DHCP 與網路路由，不用 MR-105NL 的 DHCP 功能。所以我要關閉 MR-105NL 的 DHCP 功能。選擇「設定」／「DHCP伺服器」，設定「關閉」。儲存設定。

第四步、接著指派 MR-105NL 的內部網路 IP ，這樣以後才方便進入 MR-105NL 的管理介面。選擇「設定」／「內部網路」。將內部網路的 IP 位址填入事先規劃好的 IP 。 因為 RB750G 所用的 IP 網段為 192.168.88.0/24，配合 RB750G 的設定，我規劃 192.168.88.20 給 MR-105NL 使用，所以填入 192.168.88.20。網路遮罩為 255.255.255.0。然後儲存。這時 MR-105NL 會花60秒設定新的網路狀況，設定完成後，就要改用新的 IP 重新登入 MR-105NL 管理介面。

雖然 RB750G 提供 DHCP 功能，但是在 MR-105NL 的內部網路設定頁中，我們不能指定用 DHCP 取得 IP，所以必須規劃一個靜態 IP 給 MR-105NL 。

第五步、啟用 AP 功能與連接密碼。選擇「無線網路」／「基本」。勾取「啟動」。修改 SSID 名稱為我自己用的名稱。加密類型使用安全性較高的 WPA2 PSK 。其他項目維持預設值。儲存設定。MR-105NL 的軟體功能設定到此完工。

第六步、把連接電腦與 MR-105NL 的網路線拔掉，改為連接 RB750G 與 MR-105NL。這條線的兩端都是插在 LAN 埠上。現在， MR-105NL 就變成一台延伸 RB750G 區網的集線器了。

連接方式如下圖所示。注意，RB750G 與 MR-105NL 之間的那條網路線，是接在 LAN 埠上。RB750G 的第1號埠(WAN埠)則是接 ADSL Modem；MR-105NL 的 WAN 埠則空著。所以一共還有6個埠可以接乙太網路設備。網路流量最大的電腦直接接上 RB750G。MR-105NL 則負責手機或是印表機伺服器這類不講求流量的設備。

拿起手機或筆電，透過 Wi-Fi 連接 MR-105NL。觀看手機或筆電網路資訊，就會看到它們的 IP 是由 RB750G 分配，它們也可以看到區網內的其他設備。

問與答
Q: MR-105NL 的 WAN埠 不能用嗎?

A: 它的用途是接 ADSL Modem 或光纖固網。經我測試結果，在本文的案例中，似乎不能將那個 WAN 埠調成 LAN 埠。就空著吧。

Q: 可以同時搭配 MR-105NL 的 3G 網路分享功能嗎？

A: 不行。我上面所說的設定方式，會廢掉這台的 3G 網路分享功能，完全透過另一台路由器出去。 如果你買 MR-105NL 的目的就是要用它的 3G 網路分享功能，請忽略這篇文章。

Q: 如果照這篇文章設定之後，但哪天 ADSL 斷線，我需要改回用 3G 網路分享功能的話，該怎麼辦？

A: 首先拔掉另一台網路分享器與 MR-105NL 的網路線。把網路線接在電腦與 MR-105NL 上，開啟瀏覽器登入 MR-105NL 管理介面。選擇「設定」／「DHCP伺服器」，啟動 DHCP功能。再選擇「設定」／「外部網路」，啟動外部網路，並根據你的 3G 網路供應商的設定，選擇連線模式(國內通常是「3G Mobile Internet」)，並插入你的 3G 網卡。最後儲存設定， 另一個更簡單的方法就是直接還原原廠設定。

Q: 如果設定錯誤，可以還原原廠設定嗎？

A: 按照面板指示，在接上電源的情況下，同時按住面板上的「WLAN」鈕 與「USB Eject」鈕不放，等WLAN燈閃一下後放開(大約等5秒)。接著 Status 燈號就會開始閃爍，表示 MR-105NL 正在還原系統。待 Status 燈號熄滅後就回復原廠設定了。

使用經驗

順便說一下這台的使用經驗吧。

它的型錄寫錯尺寸規格。我本來以為它跟我的 RB750G 差不多大小，可以疊在 RB750G 上面。結果買回來一看，竟然比我的 RB750G 還大，實際上是 15 公分乘11公分，不是10公分乘8公分。下圖是疊疊樂(放在桌子下，光線很差)。

管理介面很簡單，支援繁體中文，但也不會少什麼東西。像是 3G 網路的設定內容中，還可以輸入 SIM 卡的 PIN 碼，所以不必取消 PIN 碼；這是我選它的原因之一。至於本文所說的特殊用法，也是幾個步驟就能搞定。

哈哈，但我說的「簡單」是以 RB750G 的管理介面為比較基準。跟 RB750G 的管理介面比起來，家用型網路分享器的功能與管理介面都跟玩具一樣。

如果你要問我這台跑 BT 會不會斷線？TurboNAT 效能如何？是不是真的支援10000個連線數？抱歉，這些東西我通通都沒試。因為這些事我都交給專業的 RB750G 負責。

最後看無線訊號。我家是3樓透天厝，這台無線 AP 放在我二樓房間靠窗處(房子東側)。我在一樓客廳以及二樓隔四道牆的西側房間靠窗處也收得到訊號，只是西側靠窗處訊號只剩-90db 到 -92db，看 youtube 偶爾會停頓。不過我家主要是走電力線網路，無線基地台只是備用以及供手機上網。以它的價格、天線功率還有偶爾可以讓我分享 3G 網路的功能，我覺得物有所值了。
