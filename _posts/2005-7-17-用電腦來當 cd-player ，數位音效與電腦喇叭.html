談談用個人電腦來當 cd-player 的情形 (也是大部份人的情形)。用個人電腦來當 cd-player 時，最重要的是音效卡的選擇，光碟機的表現反而就比較不重要了。這裡的光碟機，指的就是一般電腦用的 CD-ROM DRiver ，而不是床頭音響的光碟機。

用光碟機當 cd-player 時，最好的效果就是另外接 DAC 跟擴大器，直接用光碟機的數位輸出端子，把訊號送給 DAC 跟擴大機，不過這樣的組合，大概 5000 元跑不掉。但不考慮外接 DAC 跟擴大機時，音效卡 就等於 DAC + 擴大器。音效卡要先將光碟機送來的數位訊號轉成類比訊號，再擴大訊號後輸出給喇叭。

根據個人的經驗，一般3,4百元的音效卡 (CMI8738, ESS-Solo1 之類的)，以及主機板內建音效裝置，因為成本考量，它們的數位轉類比的解碼晶片，以及訊號擴大線路表現都不理想。內建音效裝置最差。Creative PCI 64/128 系列則表現尚可。(聽說 Virba 128 因為省略擴大線路，所以聲音推的比較不好？)，我一位用全創龍捲風的朋友，用的是 Creative Audiy2。 PCI128 跟 Audiy2 雖然價格差好幾倍 (1200元跟4500元)，但光從立體聲-二聲道 (MP3/WMA/以及音樂CD 都屬於立體聲-二聲道) 的播放效果來看，這兩者的差異並不明顯， Audiy2 的價格貴，主要是表現在 5.1 聲道那個領域，應用在 DVD Video 上。如果是常看 DVD 電影，或是聽 DVD 歌劇時，那就選 Audiy2 吧。如果是聽音樂 (MP3/WMA/音樂CD)，那麼 PCI128 就夠了。

然而，不管音效卡多好，它上面的解碼晶片與擴大線路都好不過外接的 DAC 與擴大機 (不是電子街牌的) 。如果已經有一台價位在5000元以上，支援光纖數位輸入的擴大機時，倒是可以不管音效卡了，買那種3~400元的 CMI 晶片音效卡就夠了，直接光纖輸出到擴大機，就會很完美。

此外，還有一些小小的調整步驟。

  1. 把光碟機的音源線拔掉，不要用音源線連接光碟機跟音效卡。或者是在混音器中(音量調整)，將 CD 的音量調成靜音。
  2. CD 的音量調成靜音，怎麼聽CD?? 當然還是能聽啊，要改成用「數位音訊」，直接把數位訊號送給音效卡處理。以 windows 2000 為例，從「控制台」->「聲音及多媒體」->「硬體」-> 選擇光碟機，再按「內容」，就會看到一個「數位音效輸出」的選項了。
  3. 如果是用 winamp 聽音樂的人，最好再裝 out_ssrc 這個 plug-in 。另外再裝 in_mpg123 以及 in_cdreader 這兩個 plug-in 。 in_mpg123 可以改善 mp3 的音質， in_cdreader 可以讓 winamp 支援音樂光碟的數位音訊。
  4. 買條有 OFC Shield 的喇叭連接線來換。其實在 AV 音響界，就有很多人專門在玩線的，同時的，在電腦音效上，換玩還是有用的，因為電腦運作會產生很多電磁波，換條有屏遮 (shield) 效果的線，可以減少雜訊。一條1.8m，OFC shield的喇叭連接線，大約臺幣60~130元之間。

經過這樣的調整，再加上一組不錯的電腦喇叭，就可以壓死很多小型床頭音響了。所謂不錯的電腦喇叭，以二聲道為例，價位都在1000元以上，價位決定等級。 JS J5510 是長春款 (但我更建議 JS J707AV 或 JS J7508)。 Altec Lansing 的 AVS300 跟 VS2120 也是適宜聽音樂的電腦喇叭 (但建議 MX5020 或 XA3021)，當預算充足時，5000元以上的 MX5021 或 FX6021 才是你要抱回去的。

電腦喇叭皆為主動式喇叭，它的缺點在內建的擴大系統不好。像我的 JS J707AV ，我把它的內建擴大系統拆下來，改接我花1300元買的電子街牌擴大器，也順便把所有的喇叭連接線都換成 105 芯的線材。這般小改之後，那整個效果都升了一級。連 Yamaha MCR-E400, MCR-E300 (2萬元等級) 這種我曾經覺得不錯的床頭音響也比下去了。而我花的成本根本不到5000元。 JS J707AV 2100 元，XFire 1723 音效卡 700元，電子街牌擴大機 1300 元，線材300元。

現在有一款非常強調播放音質表現的音效播放器，叫做 foobar2000 。我現在就是用這一款，他在音樂格式以及音效設定上，有許多可以微調的選項。但對一般人來說，也可以說是多過頭了... 很多選項，我根本連碰都不碰。不想在軟體設定上花功夫的人， winamp 2.9x 或 winamp 5 依然是最佳選擇。

攝影機移動的聲音,工作人員走動的聲音,還有在最高潮時旁人倒抽一口氣的驚呼,即使只是兩聲道系統,仍然能夠清晰的表達出來. 其他如女主角的脣齒音,胸腔與喉部共鳴的聲音,甚至換氣聲,吞嚥口水的聲音,只要你的系統能夠將他再聲出來,那你的細節表現部分絕對是滿分啊!

而這張測試片,最重要的乃是在於中頻的部分. 人聲的甜美,濃郁,能夠展現出誘人的氣息乃是最難能可貴! 女主角嬌媚的轉換每一分音符,任何的對話與言談都能夠透露出些許的情感; 而在高聲吟唱時能夠讓人感受到她的愉悅,進而引發心中的悸動,這才叫做音樂性啊!

低頻的地方更無須多言,拳拳到肉的拍擊聲,讓人不只感受到'動感"而能感受到"肉感", 這才是王道啊~~~ 鬆軟的下盤,密實的質感,有如此的低頻,才是最完美的低頻; 當然速度感和衝擊力絕對不可少,如此才能讓人感受到刺激的感覺啊!

玩遊戲時，聲音的遠近感其實也很重要。聽說，好的 5.1 聲道喇叭 (如 Altec Lansing 641) 可以讓玩家在玩 CS 等遊戲時，做到「聽音辨位」，光聽腳步聲，就可以判斷對方是在前、在後，距離多遠。附帶一提，我用過的 Ozaki 喇叭，從來就沒發出過「腳步聲」，當年從 ozaki 的小白喇叭升級到 JS J5510 時，真的是被遊戲聲音嚇了一跳，「X的，以前從來不知道還有這些聲音，原來遊戲裡的 NPC 走路是有腳步聲的啊!!」。當然更不要提我放 MP3 時的感動，不知找回多少以前沒聽過的細節。

用耳機怎麼樣？

確實，耳機比較容易聽到細節。一般三、四百元價位的耳機，聲音細節的表現能力都比同價位的喇叭好。但缺點是，耳機比較不容易表現音場的深度 (聲音的遠近感)。聲音的遠近感表現，對聽古典樂、爵士樂等樂團編制的音樂時很重要。但對於流行音樂就比較沒影嚮。

我有兩個耳機，一個是300元左右的 pro 牌耳機，另一個是 700元左右的 labtec e-lite 840，在細節上的表現差不多，但 pro 牌耳機的聲音聽起來距離都一樣， e-lite 840 就比較有遠近感，可以聽出樂器的位置是遠是近。但 e-lite 840 的聲音凝聚力，還是比不上較好的喇叭，如我的 js j707av 。我形容聲音的凝聚力的方式是，雖然喇叭分別放在左、右兩邊，但你在聽時，會以為聲音是從兩顆喇叭的中間發出來的。明明中間沒東西，但就是會讓你"看到"聲音是從中間的地方發出來，而不是左右兩邊的喇叭。

除了凝聚力，還有一種東西叫「空間感」，喇叭跟耳機的表現也有差異。所謂空間感就是音樂聆聽時的空間感受。誇張的，甚至可以表現出音樂破牆而出 的感覺 (不是音量喔)。舉例來說，我常常在聽音樂時，感覺到那音樂之中，有些細節不是只出現在兩個喇叭之間的空間而己，而是從喇叭兩旁的空間傳出。也有過聲音從喇叭右側傳出，橫過喇叭之間的空間後，消失在喇叭左側的感覺，那就像是「看到」聲音橫過一樣 (那是一段電影主角騰空而起，橫越天空的背景聲) 。用耳機時，這種空間感幾乎都消失了。
