為了搭配我的小黑(X200s)筆電，前幾天去挑了一組耳擴與耳機。由於我已經有一組桌上系統，又是要搭配筆電，故只挑隨身耳擴與耳道式耳機。耳機的預算是 2000元以下。

出發前，先用自己的桌上系統聽過一遍，讓耳朵熟悉一下。桌上系統配置為:

  • Foobar2000 1.0 KS output
  • 電腦訊源: TerraTec Xfire 1723 音效卡，光纖數位輸出到 DAC。
  • 外接式DAC: 電光石火 Spitfire
  • 綜合擴大機: 王記 Wangine A2150。 Rec Out(By Pass)端子輸出到耳擴。
  • 耳機擴大機: PROTON 550。這是一台含耳機輸出孔的綜擴，只是被我拿來當耳擴罷了。我先前曾買過一台電光石光 Cute Encore 耳擴，聽過一陣後又賣掉，繼續用這台「耳擴」。
  • 喇叭: 雅瑟 Usher S520
  • 耳機: Grado SR60

筆電部份， Grado SR60 直接接上 X200s 的耳機輸出孔。當天是帶著筆電與 SR60 到店家處試聽。

試聽音樂

2010高雄國際Hi-End音響大展 紀念CD

  1. 第一首試聽曲: 巴海貝爾 - 卡農
    10秒。低音糊成一片的，可以收起來了。
  2. 第二首試聽曲: 寇思瑪 - 魔法的足跡
    聽一分鐘，女高音聽起來感覺刺耳的，也可以收起來了。
  3. 第三首試聽曲: J-L Campana - Nalu Kamusi
    多樣的鼓與金屬打擊樂器。如果鼓聲聽起來都像是同一種樂器或是電子樂，那也可以收起來了。
  4. 第四首試聽曲: 法雅 - 舞曲，人生苦短
    吉他演奏。這正好是 Grado 耳機的強項，而我已經聽慣了 Grado 的吉他聲，所以我特別挑這首來聽聽哪些耳機的聲音接近 Grado 。
  5. 第五首試聽曲: 巴哈 - 無伴奏組曲BWV1001急板
    直接附上CD內頁的說明: 沈重木塊被鎚擊所發出既堅硬又強韌的質地與反彈的尾勁，結合共鳴管與空間所形成的聲響中，蘊藏驚人的能量與複雜的波形，這裡所挑選的快速樂段，嚴苛考驗著系統的控制力與耐受力，線條是否流暢則是簡單的評斷標準。

聽完以上5首後，還會留下來的耳機，基本上都OK了。 剩下的就是各人喜好了。接著就是輕鬆的流行樂時間。

  1. 黃立行 - 吃我 (最後只好躺下來)
  2. 有坂美香 - Life Goes On
  3. Sheryl Nome starring May'n - 妖精
  4. Sheryl Nome starring May'n - ダイアモンド クレバス
  5. Sheryl Nome starring May'n - ノーザンクロス
  6. 松下優也 - Trust Me

以上試聽音樂，都是從 CD 轉成 Flac 無損音樂格式，存在筆電帶去試聽的。

耳擴

我先用 SR60 接筆電試聽耳擴。第一個試聽的是音樂潮流 HA-601 。它的硬體設計上有一個很麻煩的問題，就是音源輸入與耳機輸出孔太靠近了，SR60 的耳機接頭會卡到音源線接頭。耳道式耳機的接頭都比較小，不成問題。但大多數的耳罩式耳機的接頭都較粗，會卡到。

此外，不知道是不是太多人試聽過了，HA-601 一接上耳機，我就聽到滋滋的底噪聲。一開始我以為是 SR60 耳機接頭沒插到底的緣故。但接上店家的耳道式耳機一樣是這麼吵。因為我的耳機直接接上筆電的耳機輸出孔時，是安靜無聲的，所以我不能接受用了耳擴後反而有噪音這種事。這台耳擴就出局了。

接著我又試聽了 T-Cube T3。有高頻的滋滋聲，但比 HA-601 的狀況好許多。試聽音樂，第二首的人聲表現不錯。不過樂器的控制力略顯不足，第一首卡農的管風琴低音糊了，第三首打擊聲的鼓聲軟了。

前二個都出局了，老闆說，這樣就只剩下 NuForce Icon Mobile 這台 USB DAC 兼隨身耳擴了。這台我事先找過資料，剛好是我的預算上限，就拿來試聽。先當耳擴用，音源線從筆電耳機孔接到 NuForce Icon Mobile。耳機一接上，安靜無聲，這才是我要的耳擴啊。試聽音樂全都合格。接著再改成用 USB DAC 模式連接，毫無意外地發出了更清徹的聲音。我想，就是它了。

耳機

耳機的部份，我分別直插筆電耳機孔與接 NuForce Icon Mobile 耳擴。基本上，店家提供的耳機，我沒問型號，直接插上試聽。聲音不滿意的就還給店家。所以我只大略記得品牌，而也不知道型號。反正就是2000元以下價位。

我記得我試聽了 Sennheiser, ATH, Sony, Denon 。 Sennheiser 這一隻，很意外地在第一首卡農時就糊了，雖然其他曲目表現沒那麼慘，但還是被我捨棄了。ATH 聽第三、四首的聲音太軟，放棄。 Sony 還不錯，但第三首的鼓聲有點失真，聽得出渲染效果，第二首的女高音有點緊。 Denon 大部份表現與 Sony 那隻差不多，但沒有 Sony 的那兩個缺點。而且我覺得它的聲音挺接近 Grado SR60 的。所以我最後選擇 Denon C360。以上評語係個人主觀聽感。每個人喜好都不一樣，耳聽為憑。

原本我還想聽聽 Grado iGi ，可惜店家沒進這款。而且它的網路報價高於我的預算，所以就不堅持了。

最後，我留下來的是 NuForce Icon Mobile 與 Denon C360。
