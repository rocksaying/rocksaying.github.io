--- title: 價值、價格、貨幣數量的性質；價格不是客觀價值。 category: economics old-category: 經濟學/奧地利經濟學派 tags: [Mises,Hayek,主觀價值,邊際效用] ---

主觀價值／邊際效用的性質是「序數」。貨幣數量是「基數」。價格是「比數」，表示兩組價值與財貨數量關係的斜率。

價值是序數，財貨數量 (貨幣數量) 是基數，在交易過程中，有兩組價值與貨幣數量的訊息 (買方和賣方) ，價格理論就是在解釋這兩組訊息，如何壓縮成一個比數性質的數。而 Hayek 解釋市場機制的一個重要特性，是藉由這一個被壓縮後的數 (價格) 所傳達的訊息，而能協同市場中眾人的分工作業。

有人說「價格」可以當作是財貨的「客觀價值」，我說不是，價格的意涵並不符合「客觀的」概念。所謂「客觀價值」者，是固著在財貨的屬性，是自然科學中說的「價值」。而「主觀價值」並非指固著在財貨的屬性，而是行為人對兩財貨之重要性的「評值程序」的內涵( 奧國學派所說的「價值」，是一個動詞概念。)，是一套程序、一套函數，經濟學家慣稱為「效用函數」。

設 x,y 為兩財貨 (GOOD)，利用 E=mc^2 ，以能量型式 (E) 表示財貨的客觀價值。man 為一行為人 (ACTION)。以 C++ 語言符號表示客觀價值與主觀價值，其符示如下: 

/*
E 不能用 float 精確表示。
我這裡用 float 型態是限於 C++ 內建型別以及表示方便。
*/

class GOOD {
	float E;
} x, y;

/*
評值程序的回傳值，其實不是 int 。
用 int 型態也是限於 C++ 內建型別以及表示方便。
實際上應該是一個 boolean 值，意義為「是否取此 (X) 捨彼 (Y) 」。
*/

class ACTION {
	int subjective_value(GOOD X, GOOD Y);
	GOOD scale[] = {NULL/*省略*/};
	/*這個 scale，就是 Mises 在《人的行為 (Human Action)》
	一書中說的「等級表 (the one scale)」，
	中文版183頁，英文版119頁。*/
} man;

man.subjective_value(x,y);

透過這個符示，我可以清楚表示，何謂固著於財貨的屬性，何謂行為人心中的評值程序。兩者截然不同，價格理論並沒有發生把主觀價值轉為客觀價值的作用。

在 GOOD 中包含了 E 此一客觀價值，那麼在 subjective_value() 此一評值程序中，是否運用了 x,y 兩財貨的客觀價值呢？ (是否使用 x.E, y.E) 答案是沒有。評值程序與客觀價值無關，僅關係到兩財貨在行為人心中的等級表序列而已。價格之被誤認為是客觀價值，是源於個人沒有察覺到自己的行為對價格的影響變化，遂將之視為「既定的事實」。

價格是個社會現象，是所有參與市場活動的人對各物評值而起的相互作用所產生的。每一個人，在購買或不購買，出賣或不出賣的時候，對於市場價格的形成都發生他那一份的影響。但是，市場愈大則他的那份影響就愈小。所以，市場價格結構，自個人看來，似乎是一既定的事實，而他必須調整自己的行為以適應。 《人的行為 (Human Action)》, Mises, p.422-32
