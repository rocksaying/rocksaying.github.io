
必要檔頭:

* title
* category
* tags

選用性檔頭:

* lastupdated
* comment

comment:

預設不添加 comment 區。欲在文章後顯示 comment 區塊，應加上 comment head 並指示型態。

comment 型態有三種，不同型態使用的回應串也不同。

* shared
* single
* 主題

*shared* 為全站共用回應串，沒有特定主旨。

*single* 為單一文章專屬回應串。只有在這篇文章的回應區使用這個回應串。

如果不是 *shared* 或 *single* 這兩個關鍵字，就將 comment head 的值視為回應串的主題。
相同回應主題的文章會共用同一個回應串。
